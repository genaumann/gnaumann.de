# Salt

Mittels Salt können verschiedene Server- und Clientsysteme verwaltet werden.

Somit können z.B. Softwarepakete und Konfigurationsdateien effizient auf tausenden Systemen ausgerollt werden.

Die Software ist in Python geschrieben und wird aktuell von VMWare weiterentwickelt.
