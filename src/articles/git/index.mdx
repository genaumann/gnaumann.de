# Git

Git ist ein verteiltes Versionskontrollsystem, das es Einzelpersonen und Teams ermöglicht, Änderungen an Dateien in einem Projekt über die Zeit zu verfolgen.
Ursprünglich von Linus Torvalds im Jahr 2005 entwickelt, hat sich Git schnell zur Standardlösung für die Softwareentwicklung etabliert.
Es unterstützt das parallele Arbeiten durch Branching und Merging, bietet Mechanismen zur Fehlerbehebung und ermöglicht es Benutzern, den gesamten Verlauf eines Projekts zu betrachten oder zu einer früheren Version zurückzukehren.
